## What's new in Level 4
- Test classes live in a separate `test/` source root
- Each domain class gets its own test class (naming convention: `ClassNameTest`)
- Tests use JUnit 5 вҖ” added as a dependency via Maven or as a library in IntelliJ

## Key Concepts
- **@Test**: marks a method as a test case
- **Assertions**: `assertEquals`, `assertTrue`, `assertFalse`, `assertThrows` вҖ” verify expected outcomes
- **Arrange-Act-Assert (AAA)**: structure of every test вҖ” set up, run the thing, check the result
- **Test isolation**: each test method sets up its own state, no shared mutable state between tests
