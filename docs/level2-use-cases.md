# Level 2 — Use Cases

| # | Use Case       | Description                                                                                      |
|---|----------------|--------------------------------------------------------------------------------------------------|
| 1 | Create customer| System creates a customer with a name and ID                                                     |
| 2 | Open account   | domain.Customer can open a Savings or Checking account                                                  |
| 3 | Deposit money  | domain.Customer deposits into a specific account                                                        |
| 4 | Withdraw money | domain.Customer withdraws with overdraft protection (Checking allows -500 limit, Savings no overdraft)  |
| 5 | Check balance  | domain.Customer views balance of a specific account                                                     |
| 6 | View accounts  | domain.Customer sees all their accounts                                                                 |
| 7 | Exit           | User quits the application                                                                       |
